# Workshop Media

This folder contains workshop-specific images and media:

- `cover.png` - Workshop/country cover image for title slide
- `outputs/` - Country-specific FASTR platform outputs
- `photos/` - Workshop photos, local context images

## Usage

Reference these in custom slides or workshop.yaml:
```markdown
![Cover](media/cover.png)
![Country Output](media/outputs/anc_coverage.png)
```
